üõ†Ô∏è How to Compile LaTeX to PDF

‚ö†Ô∏è Note: PDFs are not generated yet. You need to compile the .TEX files using the instructions below.

Option 1: Batch Compile (Windows)

  1. Double-click compile_latex.bat in the courier folder
  2. Wait for compilation to complete
  3. PDFs will appear in reports/latex/

Option 2: Manual Compile

  1. Install MiKTeX or TeX Live:
    • MiKTeX: https://miktex.org/download
    • TeX Live: https://www.tug.org/texlive/
  2. Open Command Prompt
  3. Navigate to: cd c:\Users\dj-dev\Documents\courier\reports\latex
  4. Run: pdflatex filename.tex
  5. PDF will be created in the same folder

Option 3: Online Compiler (No Installation)

  1. Go to Overleaf.com (free account)
  2. Create a new project
  3. Upload the .TEX file
  4. Click "Recompile" to generate PDF
  5. Download the PDF

Option 4: VS Code Extension

  1. Install "LaTeX Workshop" extension in VS Code
  2. Open any .TEX file
  3. Press Ctrl+Alt+B to build PDF
  4. Press Ctrl+Alt+V to view PDF
