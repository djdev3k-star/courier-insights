# Analysis Directory

## Structure
- **comprehensive/**: Combined trip + payment + receipt data
- **reconciliation/**: Payment matching and verification
- **unpaid/**: Analysis of trips without payments
- **receipts/**: Receipt filing and reimbursement tracking

## Key Files
- `comprehensive/comprehensive_all_months.csv`: Master file with all trip data
- `comprehensive/comprehensive_monthly_YYYY-MM.csv`: Monthly breakdowns
- `reconciliation/three_way_reconciliation_per_trip.csv`: Trip-level reconciliation
